ajax的状态码

  • 0 未初始化 创建了ajax对象 但没有调用open
  • 1 载入 发送请求 调用了open和send
  • 2 载入完成 接收响应原始数据
  • 3 交互 将响应回来的原始数据 解析成为客户端可以调用的数据
  • 4 done 完成 通过 xhr.responseText 供客户端调用

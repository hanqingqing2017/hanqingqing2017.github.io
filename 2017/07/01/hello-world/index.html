搭建HEXO(静态)博客

安装hexo模块
npm install hexo-cli -g
生成博客项目
hexo init blog
进入目录安装依赖
cd blog && npm install
启动服务
hexo server
部署到github上

##一个账号只能部署一个,名字必须交 用户名.github.io

发布github需要一个发布到github的插件

npm install hexo-deployer-git –save
配置发布文件

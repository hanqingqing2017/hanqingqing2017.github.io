命名规范

  • 不能用汉字
  • 手字不能为数字
  • 要用有意义的单词做名称
  • 可以使用”-“”_”链接不同的单词
  • 不能使用拼音或者拼音缩写命名

    a标签

    `
    a{
    text-decoration: none;/去掉暗元素四种状态下的下划线/
    }
    a:link{}
    a:active{}
    a:visited{}
    a:hover{}

css属性的继承

  • css的继承分两种
    • 默认继承 和文字有关的css属性都是可以继承的
    • 自定义继承 inherit继承父级元素某个属性的属性值 和盒子模型、盒子位置、层级有关的属性都是不能继承的

相对路径和绝对路径

是指在某个文件中引入其他文件需要写路径

绝对路径：根据一段路径直接能找到想要找的目标

例如：IP 域名

相对路径：从一个参照物出发，去找另一个参照物

  • 寻找上一级文件的时候写../
  • 平级文件之间直接写名称
  • 向下查找的时候写“文件夹/子文件夹/子文件.后缀名”

在工作中一般使用相对路径

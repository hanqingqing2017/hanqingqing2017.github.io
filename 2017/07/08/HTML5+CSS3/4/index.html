HTML5+css3(3)

  • 行内式：通过标签属性style引入css样式，直接将css代码写在开始标签内
  • 内嵌式：将css代码写在<style></style>元素内，通常来说style元素一般放在head元素内
  • 外联式：html和css文件是独立的文件，通过link标签将css文件引入到html文件中
  • 导入式：html文件和css文件是独立的文件，通过css语句@import “url”引入一个css文件，这条语句的特定规则是：

css选择器

基本选择器

  • 标签选择器：直接将标签名当做选择器来使用 权重为1
  • 类选择器：将标签属性class的属性值当做选择器使用，在使用时需要在属性值前加一个“.”
    一个元素的类名可以有多个，每个类名之间用空格隔开
  • ID选择器：将标签属性ID的属性值当做选择器使用，在使用的时候要在属性值前面加一个“#”
    ID属性值在当前文档下只能有一个
  • 通配符选择器：选择当前文档下所有的元素权重小于1
  • 标签属性选择器：是利用标签属性的属性名和属性值来进行元素的选择

权重为10

其他选择器

  • 后代选择器（派生选择器）：父辈选择器用来划定范围，子级选择器用来添加样式。后代选择器不必将每个选择器都标出来，只需标出关键节点即可
    后代选择器的个数尽量不超过3个
  • 子级选择器：父级选择器>子级选择器（父子元素必须是相邻的关系）
  • 相邻兄弟选择器：上一个选择器+下一个相邻选择器
  • 并集选择器：又叫分组选择器，可以让不同的元素使用同一段css样式，每一个选择器用逗号隔开
    权重是独立计算
  • 交集选择器：两种属性同一个元素上的时候，可以使用这两种属性准确的找到要操作的元素，将同一个元素的不同属性结合在一起就是交集选择器
    交集选择器链接什么符号都不需要，直接写选择器即可
    PS：标签选择器和其他选择器组合的时候，标签选择器要写在前面
  • 伪类选择器：给某个元素添加状态
    基本语法: 选择器:伪类{}
    权重权重： 选择器的权重就是伪类选择器的权重 10<权重<11
  • 伪元素：通过css向html结构中，输出假元素

盒模型

组成网页的每一个元素，在页面上的表现形式都是一个矩形（所占的位置是一个矩形），这一个个的矩形都叫做盒子（因为每一个矩形都会装东西）

盒子模型 ：用来描述每一个元素之间的位置关系，和盒子本身的大小属性

元素的css属性分为两种：

  • 内置属性：元素天生自带的属性
  • 自定义属性：人为添加给元素的css属性，自定义属性的属性权重大于内置属性

    盒子模型的组成

    盒子模型由外边距，边框，内边距，和内容部分组成，其中外边距，边框，内边距都是4个方向。内容只有宽高
    每一个元素都有自己的内置样式，在编辑页面需要将这些内置样式统一重置，保证页面样式的正常展示
  • 外边距：盒子与盒子之间的距离，用css属性margin表示，margin有四个方向
    • margin-top 上外边距
    • margin-bottom下外边距
    • margin-left 左外边距
    • margin-right 右外边距
      margin值可以是数字、百分比、em、rem、并且margin值支持负数
      margin的写法：
  • margin:10px 20px 30px 40px;上右下左
  • margin:10px 20px 30px;上 左右 下
  • margin:10px 20px; 上下 左右
  • margin:10px; 四个方向的值一致
    外边距的兼容问题
  • 当上面的盒子有下外边距，下面的盒子有上外边距时，两个值只有最大值起作用
  • 子级盒子的上边距会上传给父级盒子（指的是有包含关系的所有盒子），当父级盒子没有内边距或边框的时候，就会出现这个问题
    • 解决方式：
      • 给父级元素添加 overflow:hidden;（溢出隐藏）属性
      • 给父级元素添加内边距或者是边框属性
      • 将子元素的margin-top值变为父级元素的padding-top值【最优选项】

        边框：在内边距和外边距之间，包裹盒子的四个方向的线条

  • 边框有四个方向
  • 边框可以设置颜色
  • 边框可以设置样式
    |属性|描述|
    |–|–|
    |border-top-width|上边框的宽度|
    |border-top-color|上边框的颜色|
    |border-top-style|上边框的样式|
    |border-top:1px solid red|上边框宽度样式颜色|
    |border-width:10px 20px 30px 40px;|上右下左的四个方向的边框宽度|
    |border-width:10px 20px 30px;|上 左右 下的边框宽度|
    |border:10px solid red;|四个方向的宽度 颜色 样式都一致|

transparent透明

利用边框来制作三角形

  • 内边距是指边框距离盒子内容的距离 padding有四个方向
    • padding-top 上外边距
    • padding-bottom下外边距
    • padding-left 左外边距
    • padding-right 右外边距
      padding值可以是数字、百分比、em、rem、并且padding值不支持负数
      padding的写法：
  • padding:10px 20px 30px 40px;上右下左
  • padding:10px 20px 30px;上 左右 下
  • padding:10px 20px; 上下 左右
  • padding:10px; 四个方向的值一致
  • 内容（content）分别有两个属性代表
  • 宽度width 数字 百分比 rem em
  • 高度height数字 百分比 rem em
  • 如何计算盒子的大小
    • 盒子的宽度=内容的宽度+左右内边距+左右边框宽度
    • boxwidth=width+（padding-left）+（padding-right）+(border-left-width)+(border-right-width)
    • 盒子的高度=内容的高度+上下内边距+上下边框宽度
    • boxheight=height+（padding-top）+（padding-bottom）+(border-top-width)+(border-bottom-width)

bfc块级盒子渲染模式

block formatting context
块级盒子应该如何在页面中排版？

  • 盒子从父级盒子的左上开始
  • 垂直布局
  • 如果遇到上下margin值的时候，会发生margin的折叠
  • 所有bfc渲染的盒子都是基本流（文档流）内的盒子

IFC行内盒子渲染模式

inline formatting context

  • 盒子从父级盒子的左上开始
  • 横向排布，达到父级宽度的最大值，自动折行
  • padding-top，padding-bottom，margin-top，margin-bottom可识别但不起作用

    块级元素的特点

  • 独占一行，垂直排列
  • 可以设置盒子模型的所有属性（width，height，padding，border）
  • 在不设置盒子的宽高时候，宽度是其父级盒子内容的宽度，高度是其本身内容的高度
  • 块级元素可以嵌套其他元素
    • ul、ol第一层级里面只能是li
    • dl第一层级只能是dt和dd
    • p不能嵌套块级元素

      行内元素的特点

  • 在一行显示，从左到右，达到父级元素的最大宽度的时候，自动折行
  • 不能设置宽度和高度，padding和margin的上下值
  • 宽度和高度是由本身内容的宽高决定的
  • 行内元素进行嵌套时，不要嵌套块级元素
  • 行内元素之间如果有空格或者回车，在页面上显示的时候会出现盒子与盒子之间的空隙

overflow

属性值 描述
hidden 隐藏[超出盒子的部分不显示]
auto 自动识别[哪个方向内容溢出，那么哪个放向就会出现滚动条]
scroll 出现滚动条[无论是否溢出，都会出现滚动条]
visible 默认状态
inherit 从父元素的属性上继承属性值

display 以什么样的方式来显示元素

属性值 描述
block 块级
inline 行内
inline-block 行内块级元素
list-item 列表项目[列表队列]
table 表格
table-cell 表格项

行内块级元素

  • 从父级盒子的左上角开始排列，从左到右排列，遇到父级元素最大宽度的时候，自动折行
  • 可以直接设置盒子模型的所有属性
  • 元素之间如果有空格或者回车，在页面上显示的时候会出现盒子与盒子之间的空隙
  • 同级元素之间默认垂直方向对齐方式为“基线对齐”
    基线对齐：同级元素之间的垂直对齐方式
    vertical-align：baseline；
    |属性值|描述|
    |–|–|
    |baseline|基线对齐|
    |top|顶部对齐[兄弟元素之间最高的那个元素的顶部对齐]|
    |bottom|底部对齐[兄弟元素之间最高的那个元素的底部对齐]|
    |middle|中部对齐[兄弟元素之间最高的那个元素的中部对齐]|
    |text-top|文字顶部对齐|

    如果想要改变基线对齐的方式，那么需要给给每一个改变的元素都添加上这个属性

水平对齐

这个属性要给有宽度的元素

  • 文字的水平对齐
    • 左对齐 text-align：left；
    • 居中对齐 text-align：center；
    • 右对齐 tex-align: right ；

这个属性需要给有宽度的元素

  • 盒子的水平对齐
    • 行内或行内块级元素对齐方式
      • 左对齐 text-align：left；
    • 居中对齐 text-align：center；
    • 右对齐 tex-align: right ；
  • 块级元素的对齐方式
    • 给这个块级元素宽度，并且margin:0 auto;左右值为auto即可，那么这个块级元素就会在其父级元素水平居中
